ğŸ“Œ **Project overview:**

This MLFD project aims at reproducing the flow field behind a floating wind turbine with PIV data using radial basis functions (RBFs).

This code is capable of:

- Loading processed PIV experimental data
- Computing velocity magnitude flow field using RBFs
- Computing vorticity flow field using derivatives of RBFs
- Performing parametric studies on the penalization parameter :math:`\alpha` and the RBF width :math:`\varepsilon`
